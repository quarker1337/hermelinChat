STRUDEL // HANS

Minimal REPL that survives the Artifact iframe sandbox. Click ARM once, then Play.
Play: Ctrl+Enter
Stop: Esc
State: idle
Notes
- Audio contexts require a click. If silent: click ARM again, then PLAY.
- Uses stock Strudel runtime preload as closely as possible, so sample and soundfont behavior should match strudel.cc better.
- First load can take a moment while samples and soundfonts warm up.
- Hotkeys: Ctrl+Enter = play, Esc = hush.
