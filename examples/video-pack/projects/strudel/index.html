STRUDEL // HANS

Minimal REPL that survives the Artifact iframe sandbox. Click ARM once, then Play.
Play: Ctrl+Enter
Stop: Esc
State: idle
Notes
- Audio contexts require a click. If silent: click ARM again, then PLAY.
- Paste normal Strudel patterns directly. You usually do NOT need .out() here.
- Tempo is Strudel-style: use setcpm(bpm/4) for a typical 4-beat groove.
- Samples are loaded from the fuller default dough-samples bundle; first hit can still be a little late while audio fetches.
- Full REPL visuals like ._pianoroll() / ._scope() are currently stripped in compatibility mode.
- Some GM sounds are temporarily mapped to built-in synths in compatibility mode.
- If code uses duckorbit(2), a silent orbit-2 keepalive is auto-injected.
- Hotkeys: Ctrl+Enter = play, Esc = hush.
